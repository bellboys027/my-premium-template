## ✨ Fitur Unggulan

- **Modern Architecture:** Menggunakan Next.js 15 App Router untuk rute yang lebih cepat.
- **Ultra Smooth Scroll:** Pengalaman browsing mewah dengan integrasi **Lenis Scroll**.
- **Interactive UI:** Animasi masuk (reveal) yang halus di setiap section dengan **Framer Motion**.
- **Responsive Design:** Optimal di semua ukuran layar (Mobile, Tablet, Desktop).
- **Dark Mode Ready:** Dukungan penuh untuk tema gelap (Dark Theme) yang estetik.
- **Clean Code:** Kode terstruktur rapi, menggunakan **TypeScript** untuk keamanan tipe data.

## 🛠️ Tech Stack

- **Core:** Next.js 15 (React 19)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animation:** Framer Motion
- **Scroll:** @studio-freight/react-lenis
